### 容器服务策略配置

**如果只想要给相关人员操作指定的容器服务的话可以设置不同的策略来配置**

文档地址

```
https://cloud.tencent.com/document/product/457/31556 
```

这块是地域配置

```
"resource": "qcs::tke:gz::cluster/cls-g4792rst"
------- 改成北京 -------
"resource": "qcs::tke:bj::cluster/cls-g4792rst"
```

```json
{
    "statement": [
        {
            "action": [
                "tke:Describe*",
                "tke:Check*"
            ],
            "effect": "allow",
            "resource": "qcs::tke:bj::cluster/cls-g4792rst"
        },
        {
            "action": [
                "cvm:Describe*",
                "cvm:Inquiry*"
            ],
            "effect": "allow",
            "resource": "*"
        },
        {
            "action": [
                "vpc:Describe*",
                "vpc:Inquiry*",
                "vpc:Get*"
            ],
            "effect": "allow",
            "resource": "*"
        },
        {
            "action": [
                "clb:Describe*"
            ],
            "effect": "allow",
            "resource": "*"
        },
        {
            "action": [
                "monitor:*",
                "cam:ListUsersForGroup",
                "cam:ListGroups",
                "cam:GetGroup",
                "cam:GetRole"
            ],
            "effect": "allow",
            "resource": "*"
        }
    ],
    "version": "2.0"
}
```
